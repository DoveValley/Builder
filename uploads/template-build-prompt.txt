======================  GREAT-SEO PRINCIPLES  ==========================
Apply these to every template:
  • ONE primary keyword is the focus — in the <title> (seo.seo_title), H1
    hero, the intro H2, and the URL slug. Do NOT dilute it.
  • Secondaries: weave LIGHTLY and only in the FAQ block (one long-tail per
    page, only if natural). Never in the intro/hero/features. Never stuff.
  • Uniqueness at scale = the anti-doorway defense. Each page's AI prose must
    LEAD with the single most distinctive fact about THAT city (dominant
    employer/industry/project) so no two cities open alike.
  • Entity-richness: name REAL local employers + REAL projects (from research)
    — these are entities Google ties to the city.
  • NEVER fabricate: pass rates, review counts, salaries (ranges only, from
    research), student counts, contact-hour numbers in AI prose, eligibility
    numbers. Remove any hardcoded fake aggregateRating from schema.
  • Proprietary/real signals rank high: real prices (courses.json/pricing),
    real schedule shortcode, real guarantees — AI prose supports, never stars.
  • True AI blocks for all descriptive/local content; real-DATA blocks
    (pricing, schedule, stats, testimonials) stay factual, never AI-invented.
  • Ship a few cities first, verify, THEN scale.

PHASE 3 — Archetypes (sites/[SITE_ID]/multisite/archetypes.json = per-master
